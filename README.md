# Urban Threads

Vanilla HTML/CSS/JS e-commerce storefront using Firebase Auth + Firestore.

## Run locally
```bash
npm install
npm run dev
```

The dev server prints a URL like `http://localhost:5173/`. If that port is in use, it automatically picks the next available port.

## Pages
- `/index.html` Shop
- `/product.html?id=...` Product details
- `/cart.html` Cart + checkout
- `/auth.html` Login/register
- `/account.html` Profile + default address + order history
- `/admin.html` Admin dashboard
